# **App Name**: Code Cheatsheets

## Core Features:

- Code Snippet Display: Display code snippets with syntax highlighting for Next.js, React.js, and advanced frontend concepts.
- Topic Categorization: Categorize code snippets by topic (e.g., React Hooks, Next.js Routing, Asynchronous JavaScript).
- Landing Page: Simple landing page to introduce the website and its purpose.

## Style Guidelines:

- Primary color: Deep Indigo (#6666FF) for a sophisticated and tech-oriented feel.
- Background color: Light gray (#F5F5F5) for a clean and readable interface.
- Accent color: Electric purple (#BF40BF) for highlighting important details and interactive elements.
- Headline font: 'Space Grotesk', a sans-serif, for titles to provide a modern look.
- Body font: 'Inter', a sans-serif, for descriptions and explanations to ensure readability.
- Code font: 'Source Code Pro' for displaying code snippets, maintaining clarity and distinction.
- Use clean and modern icons to represent different code categories, enhancing visual navigation.
- Card-based layout to organize code snippets and explanations in a clear and visually appealing manner.
- Subtle animations and transitions when displaying code snippets and navigating between categories to provide a smooth user experience.
